ICamera Methods
ICamera Interface See Also Send Feedback

The ICamera type exposes the following members.

Methods

  NameDescription
AutoFocus
Enable autofocus on the camera.
DisplayCameraProperties
Display a property page where the user can perform detailed settings provided by the camera driver.
Dispose
Release all resources used by the camera.
Equals
Returns whether the specific object is equal to the current object.
ManualFocus
Enable manual focus on the camera.
SetFocus
Set focus on the camera. This will enable manual focus.
SetResolution
Set the desired resolution you want the camera frames to be in. If the camera does not support the desired resolution, nothing will happen.
Snapshot
Save a snapshot of the current frame to disk.
Start
Start the camera.
Stop
Stop the camera.
ToggleRecording
Toggle video recording. If the camera is not already recording, it will start recording. If it is, it will stop.
